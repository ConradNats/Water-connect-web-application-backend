# Contributing to Water Connect Backend

Thank you for your interest in contributing to the Water Connect Backend project! We appreciate your efforts to help improve this project.

## Getting Started

1. Fork the repository
2. Clone your forked repository
3. Create a new branch for your feature or bugfix: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit your changes: `git commit -am 'Add some feature'`
6. Push to the branch: `git push origin feature/your-feature-name`
7. Submit a pull request

## Code Style

- Use consistent indentation (2 spaces)
- Follow JavaScript/Node.js best practices
- Use meaningful variable and function names
- Add comments for complex logic
- Keep functions small and focused on a single responsibility

## Commit Messages

Please write clear and descriptive commit messages:

- Use the present tense ("Add feature" not "Added feature")
- Use the imperative mood ("Move cursor to..." not "Moves cursor to...")
- Limit the first line to 72 characters or less
- Reference issues and pull requests liberally after the first line

Example:
```
Fix authentication middleware

Fixes #123
- Updated token validation logic
- Added better error messages
```

## Pull Request Process

1. Update the README.md with details of any new features or changes
2. Ensure your code follows the project's style guide
3. Write meaningful commit messages
4. Update documentation as needed
5. Test your changes thoroughly
6. Request review from project maintainers

## Reporting Bugs

When reporting bugs, please include:

- A clear description of the bug
- Steps to reproduce the issue
- Expected behavior
- Actual behavior
- Environment details (Node.js version, OS, etc.)
- Screenshots or error logs if applicable

## Feature Requests

When requesting features, please provide:

- A clear description of the feature
- The use case and why it would be beneficial
- Examples of how the feature would be used
- Any alternatives you've considered

## Questions?

Feel free to open an issue with your question or reach out to the maintainers.

Thank you for contributing!
